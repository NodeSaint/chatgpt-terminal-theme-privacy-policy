Privacy Policy

Terminal Theme for ChatGPT — Last updated: 22 March 2026

What This Extension Does

Terminal Theme for ChatGPT applies a green-on-black terminal aesthetic to chatgpt.com using CSS overrides. It changes only the visual appearance of the page.

Data Collection

We do not collect, transmit, or share any data.

The extension stores a single boolean value (theme on/off) in your browser's local storage using the Chrome Storage API. This data never leaves your device.

What This Extension Does Not Do

  • Read, intercept, or modify your conversations
  • Access any data beyond applying visual CSS styles
  • Communicate with external servers
  • Use analytics, telemetry, or tracking of any kind
  • Collect personal information

Permissions Used

  • storage — save your on/off preference locally
  • host_permissions: https://chatgpt.com/* — inject CSS on chatgpt.com only
